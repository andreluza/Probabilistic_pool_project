# species pool data


ensemble : folder with SDMs and the ensemble of these models
occ_all: range maps from IUCN (raster format)
ProbPool_* : raster  of dispersal-weighted SDMs
dispersal_*.RData: dispersal pool
env_pool_2_x_2_cellsize.RData: environment-weighted pool (result of the ensemble)
names_997spp.csv = names of the species in the pool
probabilistic_pool.RData: probabilistic pool before manually add species
probabilistic_pool_add_species.RData: probabilistic pool after manually add species present in the communities but not in the pool
